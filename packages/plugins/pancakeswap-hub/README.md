# pancakeswap-hub

AI-powered assistance for integrating and using Pancakeswap Hub.

## Skills

### hub-swap-planner

Plan token swaps through PCS Hub, PancakeSwap's aggregator API. Fetches optimal routing across multiple DEXs on BSC, presents a route summary with split breakdowns, and generates a channel-specific handoff link for the target distribution interface (PancakeSwap, Binance Wallet, Trust Wallet, or headless).

## Installation

```bash
# Claude Code
/plugin install pancakeswap-hub

# Manual
cp -r skills/hub-api-integration/SKILL.md .cursor/skills/hub-api-integration/SKILL.md
```
